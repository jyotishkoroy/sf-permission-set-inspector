# Security Policy

## Reporting a Vulnerability
If you discover a security issue, do not publish details in a public issue.
Open a GitHub issue requesting private coordination.

## Notes
- This tool is read-only (queries User, PermissionSetAssignment, PermissionSetGroupAssignment, etc.).
- Access to setup inventory can be permission-restricted in some orgs.
- The UI degrades gracefully when optional objects (e.g., Permission Set Group components) are inaccessible.
