# Permission Set Inspector

A read-only admin utility to inspect what a user has been assigned:

- Permission Sets (direct assignments)
- Permission Set Groups (group memberships)
- Quick diff between two users (A vs B)

The app is intentionally **schema-neutral**:
- No Custom Objects
- No Custom Metadata Types
- No Custom Settings
